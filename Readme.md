# Warehouse Console App (.NET 6 + MySQL)

Это консольное .NET-приложение реализует учёт паллет и коробок на складе с возможностью хранения и расчёта параметров (вес, объём, срок годности).

## Технологии
- .NET 6
- C#
- MySQL
- xUnit (юнит-тесты)

## Структура
- `Pallet` – содержит коробки, вес +30 кг, срок годности — минимальный среди вложенных коробок
- `Box` – имеет габариты, вес, дату производства или срок годности (по умолчанию 100 дней от даты производства)
- `wh.Tests` – проект с юнит-тестами (валидация размеров и сроков)

## Возможности
- Чтение паллет и коробок из базы данных
- Автоматический пересчёт веса и срока годности паллет
- Группировка и сортировка паллет по сроку и весу
- Поиск паллет с наибольшим сроком годности
